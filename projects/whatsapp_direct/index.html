Открыть в WhatsApp

Телефонный номер переводится в числовой формат автоматически (удаляются все лишние символы, остаются только цифры).

Если номер телефона начинается на 0, то телефонному номеру автоматически присваивается код страны +66.

При нажатии на кнопку "Открыть в WhatsApp", осуществляется переход по генерируемой ссылке. Эта ссылка доступна для копирования.
